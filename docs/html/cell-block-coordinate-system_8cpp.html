Converts a global cell coordinate into a (cell block + cell) coordinate.

NOTE: s32vec2 is not able to index the entire region which the Universe is able to map, hence: 'small'.

Parameters
[in]small_global_cell_coordGlobal cell coordinate to convert.
[out]resulting_cell_block_coordinatePointer to be filled out with the CellBlock coordinate containing the cell coordinate being converted.
[out]resulting_cell_coordinatePointer to be filled out with the coordinate within the resulting_cell_block_coordinate of the cell coordinate being converted.
